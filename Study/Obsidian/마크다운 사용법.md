# 마크다운을 사용해보자
- 샵(#)하고 띄우면 헤딩 작성, 6레벨 까지 가능
- 샵(#)하고 바로 작성하면 태그 생성
- 불렛 포인트는 - 를 입력하고 띄어쓰기로 사용할 수 있다
- 백스페이스로 빠져나간다
- 글씨를 굵게 강조하는 볼드는 별표* 2개로 가능하다 **이렇게
- 헤딩은 6개까지 가능
- 글머리 기호 - Tab 들여쓰기, Shift+Tab 내여쓰기, 앞에 숫자넣기
- 볼드 : **볼드**
- 기울림 : *기울림*
- 볼드+기울림 : ***볼드+기울림***
- 밑줄의 경우 HTML을 활용 : <u>밑줄</u>
- 하이라이트 : ==하이라이트==
- 삭선 : ~~삭선~~
-
