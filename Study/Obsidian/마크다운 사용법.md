# 마크다운을 사용해보자


## 기능 종류
- 태그(Tag), 중첩태그
- 제목(Headers)
- 굵게(Bold)
- 기울임, 이탤릭(Italic)
- 취소선(Strikethrough)
- 리스트(Lists)
- 링크(Links)
- 이미지(Images)
- 인용구(Blockquotes)
- 코드(Code)
- 수평선(Horizontal Rule)
- 각주(Footnotes)
- 하이라이트(Highlight)
- 체크박스(Check Box)
- 그림 크기(Images size)
- 문서 링크(Internal Links)
- 문서 임베딩(Embedding Links)
- 문단 참조(Blcok Reference)
- 강조 상자 넣기(Callout, 말풍선)
- 밑줄 - HTML 코드 활용


## 단축키
- Ctrl+B : 볼드
- Ctrl+I : 기울림
